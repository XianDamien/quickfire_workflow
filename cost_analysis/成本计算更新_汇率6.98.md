## 优化建议

### 1. 优先使用Batch API
对于批量标注任务，使用Batch API可节省50%成本。

### 2. 优化思考Token使用
思考Token占输出的79%，是成本最高的部分：
- 实时API: 125.64元/天
- Batch API: 62.82元/天

建议通过prompt优化减少不必要的思考消耗。

### 3. 混合使用策略
- 实时交互场景: 使用实时API
- 批量处理任务: 使用Batch API
- 预计可节省30-40%的整体成本

### 4. 监控汇率变化
汇率波动直接影响成本，建议定期更新计算。

## Excel文件说明

所有Sheet的备注列已添加详细计算过程，格式为：
```
Token数量 | 计算: Token数(M) × 单价(USD/M) × 汇率 = 费用(CNY)
```

示例：
```
1,044,000 tokens (53%) | 计算: 1.044M × $2.00 × 6.98 = 14.57元
```
