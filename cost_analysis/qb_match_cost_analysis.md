## 5. 注意事项

- output tokens 占总量 69%，主要来自 LLM 的 Q/A 解析和工具调用推理
- 耗时瓶颈在 API 响应（非计算），并行可线性加速
- 免费额度：qwen3.5-plus 首次开通赠送 100 万输入 + 100 万输出 tokens（90 天）
- Batch API 可享 50% 折扣，但本任务为交互式工具调用，不适用 Batch
