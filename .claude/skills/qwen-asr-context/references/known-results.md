## 2026-03-13 三方说话人分离测试

**测试音频**: `Chloe31700_2026-03-12_V1-8-D4_Stella苑.mp3` (95.7s，单声道，教师-学生抽背单词)

### 结论矩阵

| 维度 | Qwen ASR | FunASR | 豆包 |

| 英文准确度 | 最高 | 中等 | 高 |
| 说话人分离 | 不支持 | 失败(1人) | 失败(1人) |

### 核心发现

1. **说话人分离在此场景不可行**：单声道手机录音 + 教师学生音色差异不足 + 中英文频繁交替，三家声纹聚类均失败
2. **Qwen ASR 综合最优**：中英文准确度最高，system context prompt 可持续优化
3. **FunASR 的价值在时间戳**：虽然转写准确度最低，但提供了最细粒度的时间戳
4. **豆包 utterance 粒度太粗**：96 秒音频只分成 2 段，不利于细粒度分析

### 建议方向

- 放弃 ASR 层面的说话人分离
- 替代方案：LLM 标注阶段根据内容语义推断角色（教师说英文单词+例句，学生说中文释义）
- 时间戳需求可用 FunASR 补充，但主转写仍用 Qwen ASR
